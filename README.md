# Tauris - 日志处理、清洗与转发工具

## 背景



## Tauris能做什么？
* 它可以作为日志传输工具
* 它可以作为消息适配器
* 它可以作为阿里云日志服务和消息服务的客户端
* 它可以分析日志得到监控数据



Tauris在设计思想和配置方式方面借鉴了[logstash](https://www.elastic.co/cn/products/logstash)。
